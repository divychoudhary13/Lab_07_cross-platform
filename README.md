## Project Overview
This is a React Native Shopping Cart application using **Redux Toolkit**.  
It allows users to browse products, add them to the cart, view cart items, update quantities, and proceed to checkout.

---

## Features
- Product list loaded from API  
- Add products to cart  
- Update item quantities in cart  
- Remove items or clear cart  
- Shows total amount  
- Checkout flow with alerts  
- Empty cart message when applicable  
- Navigation between products and cart screens  

---

## Setup & Run
1. Install dependencies:  
npm install
2. Start Metro Bundler
npx react-native start
3. Run on Platform
- For Android
npx react-native run-android
- For iOS
npx react-native run-ios
